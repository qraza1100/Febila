# 🚀 Fundraising Landing Page (Web3 Ticketing System)

## 📌 Overview

A Vite-powered frontend for a fundraising platform, developed for a client in Nigeria. This project introduces a decentralized ticketing system utilizing blockchain and Web3 principles to ensure transparency, auditability, and trust in financial transactions.

🔗 **Live Demo:** [https://febillalanding.vercel.app/](https://febillalanding.vercel.app/)

---

## ✨ Features

* 🎟️ Blockchain-based Ticketing System
* 🔍 Transparent Transaction Tracking
* 🔐 Web3 Integration for Trust & Security
* ⚡ Fast and Optimized UI with Vite
* 📱 Responsive Design for All Devices
* 🌐 Modern Landing Page for Fundraising Campaigns

---

## 🛠️ Tech Stack

* Frontend: Vite, HTML, CSS, JavaScript
* Web3: Blockchain Integration (Smart Contract Interaction)
* Deployment: Vercel

## 🌍 Deployment

The project is currently deployed on Vercel.
To deploy your own version:

```bash
npm run build
```

Then upload the `dist` folder to Vercel or run:

```bash
vercel deploy
```
